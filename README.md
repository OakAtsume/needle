## Installation 

```bash
git clone https://github.com/user951/needle.git
chmod +x * 
needle -h

```
## Usage
Please report any problem or bug 
```bash
needle {option}
needle replace "1" "2" {file}
needle inject 1.txt 2.txt {option}
needle inject 1.txt 2.txt replace "1" "2" 
```
